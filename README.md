# eqchartview
滤波器折线图、EQ、支持增点删点

将它添加到存储库末尾的根 build.gradle 中：
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
}

# ..
dependencies {
    implementation 'com.github.adventurerDW:eqchartview:1.0.0'
}
